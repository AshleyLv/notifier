Notifier

This is a JQuery plugin which used to show notifactions=^_^=

Just click those buttons to see the

How to use it

It just need one line of code to introduce notifier into your project.
paramters:
-- type:what kind of notification would you like to show. there are five types: info, warning, danger, success, reminder, todo.
-- notification header
-- notification content
-- duration: if user doesn't close the notice, how long does the notice message display on the screen.

$('#infoBtn').click(function(){
    $.notifier('info','info header','I am info notic message','6000');
});
